# filen-py

Unofficial Python client for [Filen](https://filen.io), a zero-knowledge, client-side encrypted cloud storage service.

## Disclaimer

This project is not actively maintained at the moment. It was built for personal use and currently provides the functionality I need.

Full coverage of all Filen features is not planned (for example, Notes or Chats).

The cryptography code has not been independently audited.

I take no responsibility for any damage, data loss, security issues, or other consequences. Use this project at your own risk.
